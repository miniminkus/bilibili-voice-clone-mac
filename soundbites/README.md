# Test Sound Bites

This folder contains sample audio files for testing the voice cloning app.

## Files

- **john_cena_ch.wav** - John Cena speaking Chinese (~1.4 MB)
- **john_cena_eng.wav** - John Cena speaking English (~1.7 MB)

## How to Use

1. Run the voice cloning app
2. Drag and drop one of these files into the app
3. Enter some text
4. Click "Generate & Play" to hear the cloned voice

## Adding Your Own Samples

You can add your own audio samples to this folder. Requirements:
- Format: WAV, MP3, M4A, AIFF, or FLAC
- Duration: Maximum 10 seconds
- Quality: Clear speech with minimal background noise
- Sample rate: Any (will be converted to 24kHz automatically)

These files will be included in the repository for others to test with!
